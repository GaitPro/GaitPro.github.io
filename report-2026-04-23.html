進度摘要

學習並實作使用 Claude Code 工具進行畢業專題進度維護與網站更新。

1. 環境準備

在 Google Colab 中，我們需要先確認 Node.js 環境。Colab 通常預裝了 Node.js，但建議檢查版本。

# 檢查 Node.js 版本
!node -v
!npm -v

使用 npm 全域安裝 Claude Code：

# 安裝 Claude Code
!npm install -g @anthropic-ai/claude-code

2. 操作步驟 (Standard Operating Procedures)

  1. 設定 API 金鑰
    將指導老師提供的 API Key 設定為環境變數：
    import os
    os.environ["ANTHROPIC_API_KEY"] = "YOUR_API_KEY"
    print("API 金鑰已設定。")
  2. 啟動與驗證
    在終端機啟動 Claude 並確認目前系統狀態。
    # 啟動 claude（建議在本地端 VS Code 操作）
    claude /status
    注意：在 Colab 中直接執行互動式 CLI 可能會有輸入限制，建議在本地端開發環境（如 VS Code）操作更佳。

3. 專題維護任務 (Project Tasks)

利用 Claude Code 的自動化能力，針對畢業專題（視覺大語言模型之居家高齡者步態異常預警系統）進行以下維護：

  • 建立時間軸：紀錄開發里程碑（如：VLLM 模型選用、OpenGait 整合進度）。
  • 定期更新：設定每週進度更新機制。
  • 網站同步：確保前端展示頁面反映最新的開發狀態。
